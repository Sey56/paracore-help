Today, we at Paras Codarch are thrilled to announce the official launch of Paracore, a desktop application poised to revolutionize the way you approach Revit automation.

For too long, the process of creating and testing Revit add-ins has been cumbersome. It involves a rigid cycle of project creation, coding in Visual Studio, compiling, and manually loading and unloading DLLs in Revit. This workflow is slow, and it stifles the creative and iterative process that is so crucial to effective automation.

Paracore changes everything.

### What is Paracore?

Paracore is a desktop application that provides an integrated environment for executing and managing C# scripts for Autodesk Revit. It connects seamlessly to your active Revit session, allowing you to run scripts and see the results instantly, without ever needing to compile a single DLL or restart Revit.

Our philosophy is simple: empower users to automate Revit tasks with the full power of the Revit API, but without the traditional overhead. With Paracore, you write pure C# scripts that are executed dynamically.

### Key Features

*   **Dynamic Script Execution:** Write and run C# scripts on the fly. See your changes in Revit immediately, allowing for a rapid and interactive development cycle.
*   **Integrated Script Management:** Say goodbye to scattered script files. Paracore provides a beautiful and efficient script gallery to organize, view, and manage all your personal and team automation scripts.
*   **Seamless Team Collaboration:** Paracore is built for teams. With built-in Git integration, you can version control your scripts, collaborate with colleagues, and ensure everyone is working with the latest and greatest tools.
*   **Parameter Presets:** Save and load different sets of parameters for your scripts. This is perfect for running the same script with different configurations, saving you time and reducing errors.
