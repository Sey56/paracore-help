# Environment Setup & Hello Revit

Welcome! Today we will get your environment ready and run your very first Revit automation script using Paracore.

## 1. Prerequisites
Ensure you have the following installed:
-   **Revit** (2025 or above)
-   **Paracore Revit Add-in** (Installed via `Paracore_Revit_Installer.exe`)
-   **Paracore Desktop App** (Installed via `Paracore_Installer.msi`)
-   **Visual Studio Code** (with the C# Dev Kit extension)

## 2. Launching the Platform
Paracore works as a bridge between Revit and your code.
1.  **Open Revit**: Toggle the **Paracore Server** button in your Revit ribbon to "On".
2.  **Open Paracore Desktop**: The app will automatically detect your Revit session and show a green "Connected" status in the Top Bar.

## 3. Creating Your First Script
1.  In the Paracore Gallery, click the **"New Script"** button.
2.  Name it `HelloRevit.cs`.
3.  Click on the script card to open the **Script Inspector**.
4.  In the **Parameters** tab, click the **View Code** icon (located at the right end of the "Run Script" button) to see the default template in readonly mode.

## 4. Run the Script
1.  Click the **Run** button on the `HelloRevit` card.
