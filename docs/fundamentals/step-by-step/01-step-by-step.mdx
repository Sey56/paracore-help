# 01 - Step-by-Step: Script Sources and Structure

Follow these steps to practice managing script sources in Paracore.

## 1. Initialize a Script Source
1. Click the hamburger menu to show the **Sidebar**.
2. Click the **Initialize/Load** folder button in the **Local Sources** section.
3. Select an **empty folder** in your local storage.
4. Click **Select Folder** in the system dialog.

## 2. Load an Existing Script Source
1. Click the **Initialize/Load** button again.
2. Select a folder that already contains a `.paracore` file.
3. Click **Select Folder**.

## 3. Load Multiple Sources (Container)
1. Click the **Initialize/Load** button.
2. Select a **parent folder** that contains one or more script source subfolders.
3. Click **Select Folder**. Paracore will automatically discover all child sources.

## 4. Unloading Sources
1. To unload all inactive sources: Click the **broom button** in the Local Sources header.
2. To unload only the active source: Click the **unload active source** button (next to the source name).
