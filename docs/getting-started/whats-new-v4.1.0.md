# What's New in V4.1.0

Paracore V4.1.0 is a milestone release focused on **Mass Data Management**, **Reusable Logic**, and **Professional-Scale Stability**.

## 🏛️ Query Template Gallery
Transform your automation library into a suite of reusable BIM templates. You can now save complex filter graphs and reload them instantly when creating new scripts or sentinels.
[Learn more about the Template Gallery →](../authoring/query-builder.md#using-the-template-gallery)

## 📊 Active Data Workflows
The Results Table is now a high-performance workstation. Manage Revit data directly from the UI with single-cell editing or use the CSV engine for mass-updates in Excel.
[Explore Data Workflows →](../core-concepts/active-data-workflows.md)

## 🎯 VQB Surgical Precision
We've overhauled the Visual Query Builder to generate developer-grade C# code automatically. It now handles naming collisions and complex data types with 100% stability.
[See VQB Enhancements →](../authoring/query-builder.md)

## 🎨 Professional Ergonomics
From high-resolution typography scaling to the Viewport-Locked scroller, every part of the interface has been refined for heavy-duty professional use.
- **Maximized Canvas**: A streamlined New Script modal for complex builder workflows.
- **Sticky Integrity**: Table headers and scrollbars are permanently pinned for effortless navigation.
- **Inverted Watchtower**: A high-contrast status window for real-time sentinel monitoring.

## ⚙️ Safe-Swap Stability
Renaming, replacing, and deleting scripts is now 100% robust. Paracore automatically manages OS-level file handles to eliminate "Access Denied" errors during management operations.
