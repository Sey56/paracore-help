# First Login Guide

This guide will walk you through your initial setup and explains the local-first experience.

## Accessing Paracore

1.  **Launch the Paracore Application:** Open the Paracore desktop application (`Paracore.exe`).
2.  **Continue Offline:** In the top bar, click the **"Continue Offline"** button.
    *   This grants you immediate access to all local automation features.
    *   **Note:** Your scripts, presets, and data remain entirely on your local machine.

## What's Next?

Now that you're ready to use Paracore, you can start exploring:

*   **Explore the User Guide:** Learn how to navigate the Paracore interface and run scripts.
*   **Load Local Folders:** Use the Sidebar to access your personal collection of scripts from your local file system.
